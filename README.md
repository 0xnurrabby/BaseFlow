# BaseFlow Final Fixed

Fresh funky-boxy Base multisender with:
- proper manual wallet connect modal
- connected account modal with copy / disconnect
- no auto-connect on page load
- ETH strict multisend
- ERC20 strict multisend
- CSV upload
- line numbering fixed with scroll sync
- supports up to 10,000 pasted rows in UI
- Builder Code suffix support
- sponsored flow via wallet capabilities + paymasterService
- fallback direct transaction path

## Run
```bash
cd web
npm install
copy .env.example .env.local
npm run dev
```
